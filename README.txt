# E-Commerce Testing Framework

## Project Description
This project automates UI and API testing for an e-commerce platform using OpenCart.

## Features
- UI tests: Product search, cart management, and checkout workflows.
- Modular framework using Page Object Model.
- API validation using Python requests.

## How to run
1. python -m venv env_name
   env_name\Scripts\activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run: 
   pytest --html=reports/test_report.html
